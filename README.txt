## Overview

This bioinformatics pipeline is designed for corona variant analysis. The following tools are essential for the successful execution of the pipeline:

## Required Tools

1. **BWA (Burrows-Wheeler Aligner):**
   - Version: 0.7.17
   - Website: [BWA Website](http://bio-bwa.sourceforge.net/)

2. **Samtools:**
	- Version: 1.10 (using htslib 1.15.1)
    - Website: [Samtools Website](http://www.htslib.org/)

3. **GATK (Genome Analysis Toolkit):**
    - Version: gatk-4.2.6.1
	- Website: [GATK Website](https://gatk.broadinstitute.org/)

4. **Picard:**
	 
     - Website: [Picard Tools Website](https://broadinstitute.github.io/picard/)

5. **snpEff:**
      - Website: [snpEff Website](https://pcingola.github.io/SnpEff/)

## Usage Instructions

1. **Download and Install all the required tools:**
2. ** RUN pipeline using below command: ***
	
	bash pipeline.sh Ref_genome
